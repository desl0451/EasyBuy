各位三一同仁：
为丰富员工业余生活，享受大自然“秀山丽水 奇洞”，三一车友会将举行第三期“亲水”主题自驾游活动，现将活动安排如下：

一、活动内容

1.活动时间：2012年9月1日（周六）
2.活动地点：湖南株洲攸县酒埠江旅游风景区
3.行驶路线：公司西门——长永高速（约4公里）——长株高速（约39公里）——沪昆高速（约52公里）——醴陵东下高速（左拐直行约3公里）——左拐转106国道（往攸县方向约50公里）——左拐酒埠江大道（b42县道，约10公里）——酒埠江景区。

二、景区介绍

酒埠江旅游区位于株洲攸县东部、湘赣交界地带。它相继获得国家地质公园、国家水利风景区、国家湿地公园、国家aaaa级景区、省级森林公园、省级风景名胜 区等桂冠。酒仙湖是全国十大土坝水库之一，它集水面积610平方公里。泛舟湖上，两岸群山叠嶂，清风徐来，水波不兴，天然湖水游泳池，神清气爽，其乐无 穷；仙境乐园集游、玩、吃于一体，园区清泉水上娱乐，岩溶洞穴休闲相得益彰，地下河漂流更人浪漫、惊险刺激。

三、报名流程

1. 报名咨询：周翔宇 13507475025/610930 ，以邮件报名为准，请邮件注明所驾（拼车）车辆车型、车牌、衣服尺码等，发送至陶新建邮箱：taoxj@sany.com.cn 2.报名车辆限制20台，员工60人（三一车友会员优先，无车员工均可报名，非会员参与本次活动，可直接入会，享受会员福利）。
3.报名费用：218元/人（含景点门票，岩洞漂流、中晚餐，天然湖水游泳池、保险，文化衫，无需其它费用），车辆相关费用自理（拼车人平摊）。

四、注意事项

请所有参加自驾游同仁提前做好车辆安全保养，遵守活动纪律，统一行动，展现三一车友会良好形象！若遇下雨天气或气温骤降，活动延期举行，时间另行通知。
